Campanha FAUX#ELEVATE: O Ataque de 25 Segundos e o Uso de VBScript Massivo

Publicado por Equipe Técnica Visual Studio & Security | 24 de Março, 2026

O cenário de ameaças corporativas enfrenta uma nova operação coordenada batizada de FAUX#ELEVATE. Esta campanha utiliza engenharia social refinada para roubar credenciais e implementar mecanismos de mineração de Monero.

O Mecanismo de Infecção

Os invasores utilizam scripts VBScript de 224 mil linhas, onde apenas 226 contêm o código real, visando evitar detecção por sandboxes. O malware utiliza o Windows Management Instrumentation (WMI) para focar especificamente em sistemas empresariais conectados ao domínio.

Arsenal Técnico e Exfiltração

  • ChromElevator: Extrai dados sensíveis contornando a proteção ABE.
  • mozilla.vbs & walls.vbs: Responsáveis pelo roubo de perfis do Firefox e ficheiros do Desktop.
  • mservice.exe: Minerador XMRig que utiliza o driver WinRing0x64.sys para sequestrar a CPU.
  • Canais de Saída: Dados são enviados via SMTP para contas como olga.aitsaid@mail.ru.
Velocidade Crítica: A infecção completa ocorre em apenas 25 segundos.
Segurança Apple

Apple Lança Melhorias de Segurança em Segundo Plano

A Apple introduziu o recurso "Melhorias de Segurança em Segundo Plano" para corrigir falhas críticas como a CVE-2026-20643 no WebKit. Esta correção impede que sites maliciosos acessem dados de outros domínios.

Glossário Técnico

WMI (Windows Management Instrumentation)
Infraestrutura para gerenciamento de dados e operações em sistemas baseados em Windows.
C2 (Comando e Controle)
Infraestrutura utilizada por atacantes para enviar instruções a sistemas comprometidos.
ABE (App-Bound Encryption)
Mecanismo de proteção que vincula dados criptografados a um aplicativo específico.
Exfiltração via SMTP
Uso do protocolo de e-mail para retirar dados roubados de uma rede privada.
